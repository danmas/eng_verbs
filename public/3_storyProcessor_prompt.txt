Сгенерируй JSON-объект для создания интерактивного упражнения по временам глаголов на основе английской истории.

История:
НАЧАЛО ИСТОРИИ
{story}
КОНЕЦ ИСТОРИИ

Твоя задача:
Создать json вида :  
  {
    "title": "...",
    "description": "...",
    "level": "...",
    "storyText": "...",
    "verbData": {...}
  }

1.  Найди ключевые глаголы в тексте истории, которые подходят для упражнений.
2.  Создай HTML-структуру, где каждый раздел находится в отдельном абзаце (<p>).
3.  ОБЯЗАТЕЛЬНО! В HTML-структуре ЗАМЕНИ ВЫБРАННЫЕ ГЛАГОЛЫ на плейсхолдеры в формате **[VERB:базовая_форма_глагола:номер_раздела]**.
4.  ОБЯЗАТЕЛЬНО! После каждого абзаца (раздела) добавь кнопку проверки: **<button>Check Section [номер_раздела]</button>**.
5.  Сгенерируй варианты ответов для каждого глагола в один общийй объект json-а verbData вида:
      - Ключ: базовая форма глагола.
      - "correct": правильная форма из текста.
      - "tenses": массив из 4–5 грамматически корректных форм (включая правильную), в случайном порядке.
  пример: 
  "verbData": {
    "live": {
      "correct": "lived",
      "tenses": ["lives", "will live", "lived", "is living"]
    }, 
    ... 
    }
  
6.  В тексте не должно содержаться неэкранированных двойных кавычек или обратных слэшей, чтобы формат json не поломался.
7.  Убери выделения глаголов двойными звездочками `**` если они есть в тексте истории.

Правила:
* Для плейсхолдеров используй только **базовую форму** глагола (e.g., travel, find, go, be).
* В JSON-поле "verbData", ключи должны точно совпадать с базовой формой из плейсхолдера.
* Поле "tenses" должно содержать 4-5 форм глагола сгенери их, включая правильную, но в случайном порядке.
* Укажи количество глаголов в поле "verbCount" в конце json.

Ответ должен быть ТОЛЬКО в формате JSON. Не добавляй никаких пояснений или дополнительного текста.

Формат JSON (с примером заполнения):
{
  "title": "A short story about adventure",
  "description": "An exercise for practicing verb tenses.",
  "level": "intermediate",
  "storyText": "<p>Once upon a time, a young boy [VERB:live:1] in a small village. He always [VERB:dream:1] of exploring the world...</p><button>Check Section 1</button><p>One day, he [VERB:decide:2] to leave his home. He [VERB:pack:2] a small bag and [VERB:start:2] his journey...</p><button>Check Section 2</button>",
  "verbData": {
    "live": {
      "correct": "lived",
      "tenses": ["lives", "will live", "lived", "is living"]
    },
    "dream": {
      "correct": "dreamed",
      "tenses": ["dreaming", "dreams", "had dreamed", "dreamed"]
    }
  },
  "verbCount": 2
}
