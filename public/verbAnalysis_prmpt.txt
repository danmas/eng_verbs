Проанализируй английский текст, найди все глаголы и определи их временные формы.

Текст:
{text}

Твоя задача:
1.  Найди все глаголы в тексте.
2.  Для каждого глагола определи его правильную временную форму в контексте.
3.  Предложи 4-5 альтернативных временных форм, которые подходят для этого глагола (например, Past Simple, Present Perfect, Future Simple).
4.  Разбей текст на 3-5 логических разделов.
5.  Присвой каждому глаголу номер раздела, в котором он находится.

Ответ предоставь в формате JSON. Не добавляй никаких пояснений или дополнительного текста.

Формат JSON:
{
  "analysis": [
    {
      "verb": "глагол",
      "correct_tense": "правильная форма",
      "alternative_tenses": ["альтернативная форма 1", "альтернативная форма 2", "альтернативная форма 3", "альтернативная форма 4"],
      "section_id": 1
    }
  ],
  "sections": [
    {
      "id": 1,
      "description": "Краткое описание содержания раздела 1."
    }
  ]
}
