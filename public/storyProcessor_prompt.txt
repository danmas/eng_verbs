Сгенерируй JSON-объект для создания интерактивного упражнения по временам глаголов на основе английской истории.

История:
{story}

Твоя задача:
1.  Найди 15-25 ключевых глаголов в тексте, которые подходят для упражнений.
2.  Разбей историю на 4-6 логических разделов.
3.  Создай HTML-структуру, где каждый раздел находится в отдельном абзаце (<p>).
4.  В HTML-структуре замени выбранные глаголы на плейсхолдеры в формате **[VERB:базовая_форма:номер_раздела]**.
5.  После каждого абзаца (раздела) добавь кнопку проверки: **<button>Check Section [номер_раздела]</button>**.
6.  Сгенерируй варианты ответов для каждого глагола.

Правила:
* Для плейсхолдеров используй только **базовую форму** глагола (e.g., travel, find, go, be).
* В JSON-поле "verbData", ключи должны точно совпадать с базовой формой из плейсхолдера.
* Поле "tenses" должно содержать 4-5 форм глагола, включая правильную, но в случайном порядке.
* Укажи количество глаголов в поле "verbCount".

Ответ должен быть ТОЛЬКО в формате JSON. Не добавляй никаких пояснений или дополнительного текста.

Формат JSON:
{
  "title": "A short story about adventure",
  "description": "An exercise for practicing verb tenses.",
  "level": "intermediate",
  "storyText": "<p>Once upon a time, a young boy [VERB:live:1] in a small village. He always [VERB:dream:1] of exploring the world...</p><button>Check Section 1</button><p>One day, he [VERB:decide:2] to leave his home. He [VERB:pack:2] a small bag and [VERB:start:2] his journey...</p><button>Check Section 2</button>",
  "verbData": {
    "live": {
      "correct": "lived",
      "tenses": ["lives", "will live", "lived", "is living"]
    },
    "dream": {
      "correct": "dreamed",
      "tenses": ["dreaming", "dreams", "had dreamed", "dreamed"]
    }
  },
  "verbCount": 2
}
